## Secret Mission – Excel

Secret Mission – Excel is a project and task management system designed to organize organizational structure, manage projects, and track tasks efficiently. The system supports role-based access and provides essential planning, tracking, and reporting features.

## High-Level Scope

### Organization Structure
- Department
- Designation
- Employees
- User Roles:
  - Admin
  - Project Manager / Coordinator
  - Employee

### Project Hierarchy
- Project  
  - Module  
    - Task  

### Task Management
- Task volume / effort estimation
- Deadlines
- Sprint management
- Task assignment and self-assignment
- Task calendar view
- Gantt chart view
- Basic task reporting

## Features
- Role-based user management
- Structured organization and employee management
- Hierarchical project and task breakdown
- Sprint-based workflow
- Task visualization using Calendar and Gantt views
- Effort and deadline tracking
- Basic reporting for task progress

## User Roles
- **Admin** – Manages system settings, users, and organizational structure  
- **Project Manager / Coordinator** – Manages projects, modules, tasks, and assignments  
- **Employee** – Works on assigned or self-assigned tasks and updates progress
